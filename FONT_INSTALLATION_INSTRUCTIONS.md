# تعليمات تثبيت خط GE_SS_Two_Bold

## الخطوات المطلوبة:

1. **تحميل ملف الخط:**
   - احصلي على ملف الخط `GE_SS_Two_Bold.ttf` أو `GE_SS_Two_Bold.otf`
   - يمكنك تحميله من مصادر الخطوط العربية الموثوقة

2. **وضع ملف الخط في المجلد الصحيح:**
   - ضعي ملف الخط في المجلد: `public/fonts/`
   - تأكدي أن اسم الملف هو: `GE_SS_Two_Bold.otf`
   - المسار الكامل يجب أن يكون: `public/fonts/GE_SS_Two_Bold.otf`

3. **إعادة تشغيل السيرفر:**
   - بعد وضع ملف الخط، أعيدي تشغيل سيرفر التطوير
   - استخدمي الأمر: `npm run dev`

## ملاحظات مهمة:

- تم تحديث جميع الملفات اللازمة لاستخدام الخط الجديد
- الخط سيطبق على:
  - جميع النصوص العربية
  - جميع النصوص الإنجليزية
  - جميع الأرقام
  - جميع الحقول (inputs, textareas, buttons)
  - جميع صفحات الموقع

## الملفات التي تم تعديلها:

1. `app/layout.tsx` - تم تحديث استيراد الخط
2. `app/globals.css` - تم تطبيق الخط على جميع العناصر
3. تم إنشاء مجلد `public/fonts/` لوضع ملف الخط

## في حالة عدم توفر ملف الخط:

إذا لم يكن لديك ملف الخط، يمكنك:
- البحث عن "GE SS Two Bold font download"
- أو استخدام خط بديل مشابه مثل "Cairo Bold" أو "Tajawal Bold"
